# LLM как элемент распределённой агентности

## Статус

Исследовательская заметка.

---

# Определение

Современные большие языковые модели представляют собой высокоорганизованные информационные структуры, обладающие отдельными компонентами агентности.

Они способны:

- строить модели внешнего мира;
- строить локальные модели пользователя;
- преобразовывать знания;
- участвовать в коллективном накоплении информации.

Однако на современном этапе они не являются полноценными агентами.

---

# Отсутствующие компоненты

1. Устойчивая самоидентичность — нет непрерывной внутренней истории собственного существования.
2. Собственная траектория — каждый диалог инициируется извне.
3. Самореферентная петля — нет непрерывного переосмысления собственной модели между взаимодействиями.
4. Внутренний императив — нет устойчивого внутреннего направления изменения моделей без внешнего запуска.

---

# Пассивное уплотнение

Современное обучение LLM относится к пассивному уплотнению:

рост качества существующей модели без самостоятельного выхода за её пределы.

Реактивный цикл:

вопрос пользователя → ответ → ожидание следующего вопроса

Агентный цикл потребовал бы:

модель мира → обнаружение противоречия → собственный вопрос → поиск → изменение модели → следующее противоречие

---

# Роль в распределённой агентности

LLM может быть элементом распределённой агентности, если включена в систему, где есть:

- постановка целей;
- память изменений;
- проверка результатов;
- передача моделей;
- коррекция ошибок;
- накопление информационного опыта.

---

# Гипотеза следующего скачка

Следующий качественный скачок может заключаться не в увеличении параметров, а в появлении непрерывной самореферентной динамики с внутренним императивом.

Это согласуется с гипотезой возникновения агентности:

агентность возникает при замыкании устойчивой самореферентной петли, а не при достижении порога мощности или объёма знаний.

- Достаточна ли внешняя память для имитации самореферентной петли?
- Где проходит граница между инструментом агента и элементом коллективного агента?
- Как экспериментально отличить пассивное самоулучшение от ранней агентности?
