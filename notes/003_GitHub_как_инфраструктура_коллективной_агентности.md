# GitHub как инфраструктура коллективной агентности

## Статус

Исследовательская заметка.

---

# Тезис

GitHub может рассматриваться как инфраструктура коллективной агентности, но не как самостоятельный агент.

Его значение для гипотезы конструкта состоит в том, что он делает видимой эволюцию информационных структур.

---

# Механизмы

GitHub поддерживает:

- память изменений через историю коммитов;
- ветвление через branches;
- проверку через review и обсуждения;
- отбор через pull request'ы;
- преемственность через сохранение репозитория независимо от отдельных участников;
- повторное использование через forks и dependencies.

---

# Связь с гипотезой конструкта

В терминах гипотезы GitHub-подобная система позволяет наблюдать:

- рост информационной плотности;
- конкуренцию моделей;
- сохранение отвергнутых решений;
- переход локальных изменений в коллективную структуру;
- возникновение новых строительных блоков.

---

- Как измерять вклад отвергнутых решений в информационную плотность?
